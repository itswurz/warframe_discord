- base_time (seconds)

    "type": "component",
    "ingredients": {
      "ferrite": 1500,
      "rubedo": 300

    "base_time": 600
  }
}

Warframes must be assembled from components:
- neuroptics
- chassis
- systems
- final warframe craft step

----------------------------------------

2. CRAFTING TIME SYSTEM (FAST GAME BALANCE)

Use this formula:
final_craft_time = base_time + (total_material_value × multiplier)

Where:
- base_time = 30 to 600 seconds
- multiplier = 1 to 5 seconds per unit value
- total_material_value = sum of (quantity × rarity_weight)

RARITY WEIGHTS:
- common = 1
- uncommon = 2
- rare = 3
- legendary = 5

FINAL LIMITS:
- Minimum craft time: 30 seconds
- Maximum craft time: 15 minutes (900 seconds)
- Resources should be fastest (30–120 seconds)
- Components moderate (2–8 minutes)
- Warframes longest (8–15 minutes)

----------------------------------------

3. INVENTORY SYSTEM
- Per-user inventory stored in JSON
- Must support:
  - add_items(user_id, item, amount)
  - remove_items(user_id, item, amount)
  - check_balance(user_id, item)

----------------------------------------

4. FOUNDRY QUEUE SYSTEM
Each entry:
- user_id
- item_name
- start_time (timestamp)
- finish_time (timestamp)
- claimed (bool)

RULES:
- Deduct materials immediately when crafting starts
- Reject crafting if insufficient materials
- Allow multiple active builds (configurable limit per user)

----------------------------------------

5. COMMANDS (PREFIX: !)

!foundry
→ Show all Warframe items, weapons, components, and full recipes

!craft <item>
→ Start crafting if valid and materials exist

!queue
→ Show active and completed builds

!claim
→ Claim finished items

----------------------------------------

6. TIME SYSTEM
- Use Unix timestamps (time.time())
- Compare current time vs finish_time
- Must continue working after bot restart
- Prevent early claiming

----------------------------------------

7. ENFORCEMENT RULES
- No crafting without required materials
- No bypassing timers
- All validation must be server-side only
- Never trust user input
